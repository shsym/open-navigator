### 1. Tree
```bash
├── cluster.yml
├── controllers
├── ingress
├── namespace.yml
├── pods
├── services
└── volumes
```

| File (Dir) | Description |
| ---- | ------------ |
| cluster.yml | 클러스터 단위 설정 |
| namespace.yml | 네임스페이스 단위 설정 |
| ingress | 인그레스 객체 |
| services | 서비스 객체 |
| controllers | 컨트롤러 객체 |
| pods | 컨테이너 설정 객체 |
| volumes | 스토리지 설정 객체 |
